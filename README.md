# My WebSocket App

A lightweight, real-time Node.js WebSocket server paired with a clean HTML/JavaScript frontend client. The application establishes a persistent connection to send and receive instantaneous messages.

## 📁 Project Structure

- `server/` - Node.js backend using the `ws` library
- `app/` - Frontend HTML client interface

## 🚀 Getting Started

### 1. Prerequisites
Make sure you have [Node.js](https://nodejs.org/) installed on your machine.

### 2. Setup the Server
Navigate into the server directory, install dependencies, and start the node server:
```bash
cd server
npm install
node index.js
